# 🚀 About

**Galactic Journey Game** is a browser-based space adventure game developed using **HTML, CSS, and JavaScript**.

Players control a spaceship and explore a large universe while collecting valuable items, avoiding enemies, managing fuel, and progressing through increasingly challenging levels.

The game combines exploration, survival, and score-based gameplay with a modern interface to deliver an enjoyable space adventure experience.

---

# ✨ Features

* 🚀 Spaceship navigation
* 🌌 Large explorable game world
* 👾 Multiple enemy types
* 🪐 Collectible space items
* ⛽ Fuel management system
* 💣 Bomb ability
* 🗺️ Mini-map navigation
* 🏆 Score and level progression
* 📱 Mobile touch controls
* 🎵 Background music & sound effects
* 🎨 Modern user interface
* ⚡ Smooth gameplay

---

# 🎮 Gameplay

Travel through space, collect valuable resources, avoid dangerous enemies, manage your fuel wisely, and survive for as long as possible to achieve the highest score.

Each level introduces new challenges, encouraging players to improve their skills and reaction speed.
